# CFD Model Validation

This project compares experimental and simulation velocity data using interpolation, error analysis, and polynomial regression fitting.

## Features
- Interpolates simulation data onto experimental time points
- Calculates average error, RMSE, and maximum error
- Identifies and visualizes the point of maximum error
- Applies polynomial regression to fit experimental trend

## Output
The project generates a comparison plot showing:
- Experimental data
- Simulation data
- Interpolated simulation curve
- Polynomial regression fit
- Maximum error location
